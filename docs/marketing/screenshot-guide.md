# Screenshot Guide

Main goal:
- sell the app's real differentiator: no moderator is needed and everyone gets to play.

Recommended screenshot order for Google Play:
1. "No moderator needed"
   Suggested screen: `reveal_pass_to_player` or a clean setup screen.
2. "Everyone gets to play"
   Suggested screen: reveal intro / pass-the-phone flow.
3. "Private role reveals"
   Suggested screen: hold-to-reveal or role reveal screen.
4. "Fast setup for groups"
   Suggested screen: setup with players, impostors and categories.
5. "Short rounds, quick replay"
   Suggested screen: round ready or result.
6. "Bluff. Vote. Reveal."
   Suggested screen: result.

Practical rules:
- Each screenshot must work on its own.
- Do not split one message across multiple images.
- Avoid panorama-style sequences where one screenshot depends on the next.
- Keep typography large and copy short.
- Avoid claims like "No ads", "Best", "Top", "Free" or CTAs like "Download now".
- Keep the feature graphic separate from the screenshots.
