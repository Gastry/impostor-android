# Play Store EN

## Suggested name
El Infiltrado: Group Game

## Short description
Offline hidden-role game for groups using one shared phone.

## Long description
El Infiltrado is a fast social deduction game for friends and family. Everyone plays in person and only one phone is needed.

Set up a round in seconds:
- 3 to 12 players
- 1 or 2 infiltrators
- category based word pool
- quick mode and advanced options

How it works:
1. Configure the round.
2. Pass the phone privately so each player reveals their role.
3. Discuss, bluff and vote.
4. Reveal the secret word and infiltrator at the end.

Play offline. No account required.

## Feature bullets
- Safe hold-to-reveal interaction
- Local round history
- Recent-word repetition protection
- Theme and language settings
- Designed for fast back-to-back rounds
