# Notas De Diseno

## Direccion visual
- Estilo limpio con contraste alto y acento party misterioso.
- Fondo degradado suave + tarjetas elevadas.
- Tipografia de sistema para legibilidad.
- CTA principal dominante en Home.

## Decisiones UX
- Revelado seguro con gesto de mantener pulsado.
- Flujo minimizado de taps por jugador:
  1) pasar movil,
  2) mantener para revelar,
  3) ocultar y pasar.
- Opciones avanzadas colapsadas en Setup para no saturar.
- Revancha inmediata en pantalla de resultado.

## Accesibilidad
- Targets tactiles amplios.
- Contraste alto.
- Opcion de reducir movimiento.
- Contenidos clave en texto, no en imagen.

## Contenido
- Dataset local ampliable por JSON.
- Filtro de palabras recientes para mejorar rejugabilidad.
